# pyJsonParser

This is a small command line utility I build to parse json files. Beacause I don't like reading configutaion text files. JSON is <3.

# How to use
```
usage: pyJSONParser.py [-h] (-k K | -v V | -t T) [--validate] file                                                                                                                                                                                positional arguments:                                                                                                      file        Input Json file.                                                                                                                                                                                                                    optional arguments:                                                                                                        -h, --help  show this help message and exit                                                                              -k K        Get Keys                                                                                                     -v V        Get Values                                                                                                   -t T        Get type                                                                                                     --validate  Validate input json file and print errors
```
